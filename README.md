# Промышленный ML: Прогнозирование температуры стали и симуляция техпроцесса

## 📌 Описание проекта
Разработана прогностическая модель машинного обучения для металлургического комбината «Стальная птица», которая с высокой точностью предсказывает температуру сплава на этапе обработки стали. Главная цель проекта — минимизация потребления электроэнергии дуговыми сталеплавильными печами и оптимизация производственных расходов. Финальная модель адаптирована для использования в качестве цифрового симулятора технологического процесса при планировании производства.

## 🎯 Бизнес- и инженерные задачи
* **Оптимизация энергопотребления:** Снижение расходов на электричество за счет точного контроля и прогнозирования циклов нагрева.
* **Имитация техпроцесса (Цифровой двойник):** Построение надежной ML-модели, способной симулировать термодинамические показатели до начала фактической плавки.
* **Глубокий Feature Engineering:** Анализ сложных физико-химических параметров производства для учета нелинейной динамики технологического процесса.

## 🛠 Стек технологий и инструменты
* **Язык программирования:** Python
* **Обработка данных:** Pandas, NumPy
* **Машинное обучение:** Scikit-Learn, LightGBM / CatBoost / XGBoost *(укажите библиотеку, которую вы использовали)*
* **Оптимизация:** Optuna *(или другой инструмент, если подбирали гиперпараметры)*
* **Контекст данных:** Данные промышленных датчиков IoT, агрегация временных рядов

## 📊 Проектирование признаков и этапы реализации (Workflow)
1. **Контроль качества данных (Data Quality):** Очистка данных от аномалий зашумленных датчиков, удаление незавершенных или прерванных циклов нагрева, выделение валидных исторических батчей.
2. **Расчет термодинамических признаков:** Вычисление полной и активной затраченной электроэнергии на основе логов мощности дуговых печей и длительности фаз нагрева электродов.
3. **Учет физико-химических факторов:** Агрегация признаков по объему сыпучих материалов, используемых ферросплавов (присадок) и объему продувки сплава инертным газом для точного учета изменений в составе стали.
4. **Выравнивание временных рядов:** Выделение первой (стартовой) температуры плавки и финального целевого замера для обучения прогностического алгоритма.

## 📈 Ключевые результаты проекта
* Разработан автоматизированный пайплайн предобработки зашумленных логов промышленных датчиков.
* Обучена устойчивая предиктивная модель с оптимизированными гиперпараметрами, готовая к интеграции в качестве цифрового симулятора плавки.
